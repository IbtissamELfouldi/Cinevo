Why Cinévo?

Cinévo offers a clean, responsive, and fast user experience for discovering movie details from the OMDb API. With real-time search, beautiful cards, and minimal UI, it's the perfect tool for film lovers.

Features You'll Love

  • Keyword search with instant results
  • Movie posters, titles, and release years
  • Built with HTML, CSS, JavaScript + Axios
  • No login, no ads – just search and enjoy!
