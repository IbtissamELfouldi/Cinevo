Cinévo

A real-time movie search app using the OMDb API

About This Project

Cinévo is a simple, elegant, and responsive front-end web application that allows users to search for movies using the public OMDb API. Built with HTML, CSS, and JavaScript, it offers an intuitive interface where results are displayed as beautifully styled movie cards.

Features

  • Real-time movie search by keyword (e.g., “Batman”)
  • Display results with poster, title, and release year
  • API integration using Axios
  • Fully responsive and clean UI

Technologies Used

HTML5 CSS3 JavaScript ES6+ Axios OMDb API
